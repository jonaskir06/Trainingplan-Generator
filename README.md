## Trainingsplan-Generator (PStA OOP)

Dieses Projekt ist eine **Prüfungsstudienarbeit (PStA)** im Fach Objektorientierte Programmierung.  
Ein **Trainingsplan-Generator in Java**, der basierend auf Erfahrungslevel und Trainingsziel automatisch einen realistischen Wochen-Trainingsplan erstellt.  
Ziel des Projekts war die praxisnahe Anwendung zentraler **OOP-Konzepte** in einem zusammenhängenden Anwendungsfall.
